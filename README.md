Learning about Language Models (LLMs and ChatModels) and Embedding Models 
#

**Language Models** : Language Models are AI system designed to process, generate and understand natural language text.

i) LLMs: 
LLMs are general purpose models that are used for raw text generation. They take plain text as input and return a string (plain text). 
ii) ChatModels:
Chat Models are language models that are specialized for conventional tasks. They take a sequence of messages as inputs and return chat messages as outputs. More common than LLMs in the present days.
